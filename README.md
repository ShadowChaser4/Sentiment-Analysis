## Sentiment Analysis 
--- 
> A Deep Learning Based sentiment analysis project built using LSTM
 
Project built using Python version ```3.10.9``` 

To run the project 

Create a virtual environment and then

```bash
pip install -r requirements.txt
streamlit run app.py
```
